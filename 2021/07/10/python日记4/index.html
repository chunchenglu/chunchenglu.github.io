1.正则表达式re

首先需要引用：re
注：表达式中的r’’用在pattern之前，表示单引号中的字符串为原生字符，不会进行任何转义

import re

re.match #从开始位置开始匹配，如果开头没有则无
例如：

re.match(r'l','liuyan1').group()  #返回l
re.match(r'y','liuyan1')  #返回None

re.search #搜索整个字符串
例如：

re.search(r'y','liuyan1').group() #返回y

re.findall #搜索整个字符串，返回一个list
例如：

re.findall(r'y','liuyany')) #返回["y","y"]

正则表达式（文章链接）可以包含一些可选标志修饰符来控制匹配的模式。

re.I 使匹配对大小写不敏感
re.L 做本地化识别（locale-aware）匹配
re.M 多行匹配，影响 ^ 和 $
re.S 使 . 匹配包括换行在内的所有字符
re.U 根据Unicode字符集解析字符。这个标志影响 \w, \W, \b, \B.
re.X 该标志通过给予你更灵活的格式以便你将正则表达式写得更易于理解

正则表达式实例

详细匹配可查询上述文章链接，以下描述选取常用匹配模式进行举例

字符类

[^aeiou]匹配除了aeiou字母以外的所有字符
[^0-9]匹配除了数字外的字符
[0-9]匹配任何数字。类似于 [0123456789]

特殊字符类

.匹配除 “\n” 之外的任何单个字符。要匹配包括 ‘\n’ 在内的任何字符，请使用象 ‘[.\n]’ 的模式。

import re
name="zhangsan,张三,李四,xiaohong12382/r/n/t" 
name2 =re.findall(r'.',name)
print(name2)

#输出结果为：
['z', 'h', 'a', 'n', 'g', 's', 'a', 'n', ',', '张', '三', ',', '李', '四', ',', 'x', 'i', 'a', 'o', 'h', 'o', 'n', 'g', '1', '2', '3', '8', '2', '/', 'r', '/', 'n', '/', 't']

.*匹配除 “\n” 之外的任何多个字符。要匹配包括 ‘\n’ 在内的任何字符，请使用象 ‘[.\n]’ 的模式。

import re
name="zhangsan,张三,李四,xiaohong12382/r/n/t" 
name2 =re.findall(r'.*',name)
print(name2)

#输出结果为：
['zhangsan,张三,李四,xiaohong12382/r/n/t', '']

\d匹配一个数字字符。等价于 [0-9]。

import re
name="zhangsan,张三,李四,xiaohong12382/r/n/t" 
name2 =re.findall(r'\d',name)
print(name2)

#输出结果为：
['1', '2', '3', '8', '2']

\d+匹配多个数字字符。等价于 [0-9]。

import re
name="zhangsan,张三,李四,xiaohong12382/r/n/t" 
name2 =re.findall(r'\d+',name)
print(name2)

#输出结果为：
['12382']

\D 匹配一个非数字字符。等价于 [^0-9]。

import re
name="zhangsan,张三,李四,xiaohong12382/r/n/t" 
name2 =re.findall(r'\D',name)
print(name2)

#输出结果为：
['z', 'h', 'a', 'n', 'g', 's', 'a', 'n', ',', '张', '三', ',', '李', '四', ',', 'x', 'i', 'a', 'o', 'h', 'o', 'n', 'g', '/', 'r', '/', 'n', '/', 't']

\D+匹配多个非数字字符。等价于 [^0-9]。

import re
name="zhangsan,张三,李四,xiaohong12382/r/n/t" 
name2 =re.findall(r'\D+',name)
print(name2)

#输出结果为：
['zhangsan,张三,李四,xiaohong', '/r/n/t']

\s 匹配任何空白字符，包括空格、制表符、换页符等等。等价于 [ \f\n\r\t\v]。

import re
name="zhangsan,张三,李四,xiaohong12382   /r/n/t"
name2 =re.findall(r'\s',name)
print(name2)

#输出结果为：
[' ', ' ', ' '] 
#因为字符串中包含了三个空格如果匹配改为\s+
则输出结果为：
['   ']

\S+ 匹配任何非空白字符。等价于 [^ \f\n\r\t\v]。

import re
name="zhangsan,张三,李四,xiaohong12382   /r/n/t"
name2 =re.findall(r'\S+',name)
print(name2)

#输出结果为:
['zhangsan,张三,李四,xiaohong12382', '/r/n/t']  #把多余的空格去除了

匹配模式

书写正则表达式例如：
如果想要匹配name中的李四方式1：

import re

name="zhangsan,张三,李四,xiaohong12382/r/n/t" # 如果想要匹配name中的李四方式1：
name2=re.compile(r'张三,(.*?),xiaohong2')
name3=re.findall(name2,name)
print(name3)

#运行结果：
['李四']

如果想要匹配name中的李四方式2：

import re

name="zhangsan,张三,李四,xiaohong12382" # 如果想要匹配name中的李四方式2：
name2 =re.findall(r'李四',name)
print(name3)

#运行结果：
['李四']

字符串的替换和修改

re 模块还提供了对字符串的替换和修改函数，他们比字符串对象提供的函数功能要强大一些。这几个函数是：

sub ( rule , replace , target [,count] )
subn(rule , replace , target [,count] )

在目标字符串中规格规则查找匹配的字符串，再把它们替换成指定的字符串。你可以指定一个最多替换次数，否则将替换所有的匹配到的字符串。第一个参数是正则规则，第二个参数是指定的用来替换的字符串，第三个参数是目标字符串，第四个参数是最多替换次数。这两个函数的唯一区别是返回值。

sub 返回一个被替换的字符串
subn返回一个元组，第一个元素是被替换的字符串，第二个元素是一个数字，表明产生了多少次替换。
例：将下面字符串中的 ’dog’ 全部替换成 ’pig’

import re
s='I have a dog , you have a dog , he have a dog'
re.sub( r'dog','pig',s )
#输出结果为;
I have a pig , you have a pig , he have a pig

如果我们只想替换前面两个，则:

import re
s='I have a dog , you have a dog , he have a dog'
re.sub( r'dog','pig',s,2 )

#输出结果为：
I have a pig , you have a pig , he have a dog

或者我们想知道发生了多少次替换，则可以使用 subn

import re
s='I have a dog , you have a dog , he have a dog'
re.sub( r'dog','pig',s )

#输出结果为：
('I have a cat , you have a cat , he have a cat ,3')

2.BeautifulSoup

1）什么是 BeautifulSoup？

BeautifulSoup是 python 的一个库，最主要的功能是从网页抓取数据，它是一个工具箱，通过解析文档为用户提供需要抓取的数据，因为简单，所以不需要多少代码就可以写出一个完整的应用程序。

BeautifulSoup 自动将输入文档转换为 Unicode 编码，输出文档转换为 utf-8编码。BeautifulSoup 支持 Python 标准库中的 HTML 解析器，还支持一些第三方的解析器，如果我们不安装它，则 Python 会使用 Python 默认的解析器，lxml 解析器更加强大，速度更快，推荐安装。

2）Python 标准库

BeautifulSoup(markup, “html.parser”)
Python 的内置标准库
执行速度适中
文档容错能力强
Python 2.7.3 or 3.2.2)前的版本中文档容错能力差

3) lxml HTML 解析器

BeautifulSoup(markup, “lxml”)
速度快
文档容错能力强
需要安装 C 语言库

4) lxml XML 解析器

BeautifulSoup(markup, “lxml”)，BeautifulSoup(markup, “xml”)
速度快
唯一支持 XML 的解析器
需要安装 C 语言库

BeautifulSoup四大对象种类

了解完解析方式我们再来看看BeautifulSoup的四大对象种类
1.Tag
2.NavigableString
3.BeautifulSoup
4.Comment
Tag 是什么？通俗点讲就是 HTML 中的一个个标签，例如：

<title>The Dormouse's story</title>
<a class="sister" href="http://example.com/elsie" id="link1">Elsie</a>

我们可以利用 soup加标签名轻松地获取这些标签的内容,例如：

soup1 =BeautifulSoup(html,'lxml')
#print (soup1.prettify())
print(soup1.title)   #只查找第一个
print(soup1.head)
print(soup1.a)
print(soup1.p)

结果输出为：

不过有一点是，它查找的是在所有内容中的第一个符合要求的标签。但可以不可查找所有符合条件的标签那？当然可以！例如：

搜索文档树
find_all( name , attrs , recursive , text , **kwargs )

find_all() 方法搜索当前tag的所有tag子节点,并判断是否符合过滤器的条件，name 参数可以查找所有名字为 name 的tag,字符串对象会被自动忽略掉

soup1 =BeautifulSoup(html,'lxml')
#print (soup1.prettify())
soup2=soup1.find_all("a")
print(soup2)

soup1 =BeautifulSoup(html,'lxml')
#print (soup1.prettify())
soup2=soup1.find_all("a")
soup2=str(soup2)
a=re.findall(r'<a class="sister" href="(.*?)" id=.*?</a>',soup2)
print(a)

第二种方式为先遍历检索到的内容，然后用get获取内容例如：


soup1 =BeautifulSoup(html,'lxml')
